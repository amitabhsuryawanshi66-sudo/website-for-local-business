# 21st.dev / Magic-Style Workflow Reference
Source: https://21st.dev

## Component Variation Workflow
For every major UI section, don't implement the first idea. Internally evaluate 3 directions:
1. **Premium Minimal Luxury**: High whitespace, thin borders, understated typography.
2. **Bold Instagram Brand**: Massive headings, high-contrast accents, vibrant energy.
3. **Editorial Boutique**: Offset layouts, varied typography scales, storytelling focus.

## Implementation Strategy
- **Compare & Select**: Choose the strongest direction based on the client's specific niche (e.g., Editorial for Bridal, Bold for Car Detailing).
- **Avoid Generic Blocks**: Never use standard 3-column "feature boxes" with simple icons.
- **Reference Categories**:
    - **Heros**: Asymmetric compositions with high-quality visual leads.
    - **Packages/Pricing**: Clear hierarchy, prominent "Most Loved" badges.
    - **Testimonials**: Large typography, tactile card designs.
    - **CTAs**: High-conversion placement with prefilled WhatsApp paths.
