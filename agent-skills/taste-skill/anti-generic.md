# Taste Skill: Anti-Generic Design
Source: https://github.com/Leonxlnx/taste-skill

## The Goal
Stop generating "boring, generic slop" and start producing high-end, human-feeling interfaces.

## Anti-Slop Checklist
- **Avoid Predictable Grids**: Use asymmetric layouts, bento grids, or offset compositions to create visual tension.
- **Stronger Composition**: Focus on editorial visual flow. Sections should feel like "chapters" in a brand story.
- **Memorable Sections**: Every section should have a unique visual hook or rhythm.
- **Premium Brand Feel**: Use sophisticated color palettes and high-quality typography that matches the niche.
- **Tactile UI**: Use subtle texture (noise, grain, grid) or glassmorphism to add depth and life to digital surfaces.

## Dials for Design
1. **Design Variance**: How much to experiment with layout (Low = Clean, High = Experimental).
2. **Motion Intensity**: How much movement is in the UI (Low = Hover only, High = Scroll-driven magnetism).
3. **Visual Density**: Information per viewport (Low = Spacious, High = Dashboard).
