# UI UX Pro Max: Design Intelligence
Source: https://github.com/nextlevelbuilder/ui-ux-pro-max-skill

## Key Features
1. **Industry-Specific Reasoning**: Analyze the business niche (e.g., Beauty, Photography) and apply matching design patterns.
2. **Coherent Design Systems**: Define clear rules for Pattern, Style, Colors, Typography, and Effects before building.
3. **Conversion-Optimized Layouts**: Place CTAs above the fold and repeat them strategically after social proof.
4. **Social Proof-Focused**: Prioritize trust elements (Reviews, Ratings, Verification badges) early in the user journey.

## Pro Max Style Recommendations for Services
- **Best Patterns**: Hero-Centric + Social Proof.
- **Best Styles**: Soft UI Evolution, Glassmorphism, or Minimalist Swiss.
- **Accessibility**: Mandatory WCAG AA compliance and `prefers-reduced-motion` support.
- **Checklist**: Cursor-pointer on all clickables, smooth transitions (150-300ms), 4.5:1 minimum contrast.
