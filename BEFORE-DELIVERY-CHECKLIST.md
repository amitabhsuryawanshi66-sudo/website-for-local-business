# ✅ Before Delivery Checklist

Run through this list before sending the final link to your client to ensure a 5-star experience.

## 1. Visual & Content Check
- [ ] **Business Name:** Correct spelling throughout.
- [ ] **Niche/Tagline:** Matches the client's branding.
- [ ] **Images:** All 6 gallery images are high quality and relevant.
- [ ] **Pricing:** Prices match the client's latest rate card.
- [ ] **Location:** Correct area name and Google Maps embed.

## 2. Technical Check
- [ ] **WhatsApp Link:** Click the "Check Availability" button—does it open the correct WhatsApp chat?
- [ ] **WhatsApp Message:** Does the prefilled message sound natural?
- [ ] **Instagram Link:** Does the footer Instagram icon lead to their profile?
- [ ] **Mobile Responsive:** Check the site on your phone.
- [ ] **Speed:** Does it load instantly?

## 3. Branding
- [ ] **Primary Color:** Does the brand color match their logo/vibe?
- [ ] **Accent Color:** Are the buttons high-contrast and easy to see?

## 4. Deployment
- [ ] **Custom URL:** Is it deployed on a clean URL (e.g., `clientname.netlify.app`)?
- [ ] **Favicon:** (Optional) Replace the default favicon with their logo.
