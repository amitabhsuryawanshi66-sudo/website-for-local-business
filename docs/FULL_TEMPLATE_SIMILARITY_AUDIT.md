# Full Template Similarity Audit (V5 vs Legacy)

This audit identifies where V5-old still resembles the generic V1/V2/V4 template family.

| Element | Status | Generic/Reused Pattern | Fix Required |
| :--- | :--- | :--- | :--- |
| **Button Style** | ❌ FAILED | Standard rounded pills with same hover effects. | Rebuild with tactile, niche-native shapes and micro-interactions. |
| **Card Style** | ❌ FAILED | Uniform rounded rectangles. | Use varied card archetypes (Decision, Proof, Intent). |
| **Sticky CTA** | ❌ FAILED | Centered floating bar/capsule. | Redesign as a "Sticky Decision Bar" that updates with state. |
| **Hero Structure** | ❌ FAILED | Centered headline or split image/text. | Implement asymmetric editorial compositions. |
| **Layout Rhythm** | ❌ FAILED | Section → Card Grid → Section. | Introduce intent-based flows and asymmetric white space. |
| **Offer Section** | ❌ FAILED | Standard 3-column "Pricing Cards". | Convert to an "Interactive Offer Board" with user-choice logic. |
| **Typography** | ❌ FAILED | Default Tailwind Inter-centric feel. | Overhaul with strong editorial display hierarchy. |

## 🛡️ Rejecting the Template Look
- **REJECT:** Generic white/light-gray backgrounds for everything.
- **REJECT:** "Learn More" / "Get Started" buttons.
- **REJECT:** Symmetric grids.
- **REJECT:** Placeholder copy like "Premium Product Standards."

**Conclusion:** A total design-language replacement is required for V5 to meet pitch-ready standards.
