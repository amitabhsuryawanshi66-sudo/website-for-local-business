# Component Variation Decisions (V5)

Internal evaluation log for V5 components based on 21st.dev / Magic-style workflow.

| Component | Considered Variations | Selected Direction | Reason |
| :--- | :--- | :--- | :--- |
| **IntentHero** | A: SaaS Split, B: Centered Typographic, C: Editorial Asymmetric | **C: Editorial Asymmetric** | Maximizes visual tension and separates the brand from low-cost templates. |
| **DecisionDock** | A: Simple Nav, B: Segmented Pill, C: Tactile Floating Dock | **C: Tactile Floating Dock** | Superior thumb-zone ergonomics and clearer information scent for intent selection. |
| **OfferBoard** | A: Pricing Grid, B: Feature List, C: Decision Cards | **C: Decision Cards** | Helps users self-segment into specific service tiers based on their needs. |
| **VisualNarrative** | A: Standard Grid, B: Masonry Collage, C: Horizontal Slider | **B: Masonry Collage** | More "Instagrammable" and allows for diverse image aspect ratios. |
| **ObjectionResolver**| A: FAQ List, B: Card Wall, C: Interactive Briefing | **C: Interactive Briefing** | Actively resolves buying barriers near conversion points rather than burying them at the bottom. |
