- **Editorial Composition:** High visual tension and asymmetric layouts that separate your clients from cheap AI templates.

## 🚀 Creating a New Client Site (The 20-Min Workflow)

1.  **Duplicate Data:** Open `src/data/clientConfig.js` and add a new entry for your client (or edit an existing niche).
2.  **Brand Tokens:** Set `colors.primary` and `colors.accent`. The Pro Max CSS system handles the rest.
3.  **Content Swap:** Update the `hero`, `about`, and `services` objects with the client's specific copy and pricing.
4.  **Portfolio:** Paste 6 high-quality image URLs into the `gallery` array.
5.  **WhatsApp Sync:** Update the `whatsappNumber` and custom prefilled messages.
6.  **Build & Launch:**
    ```bash
    npm run build
    ```
    Drag and drop the `dist/` folder to Netlify/Vercel.
